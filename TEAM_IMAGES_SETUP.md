# Team Images Setup Guide

## Folder Location
Place the 4 team member images in: `public/team/`

## Required Images

Please add the following images with these exact filenames:

1. **yahya-abdul-rahim-bey.jpg** (or .png, .webp)
   - **Name:** Yahya Abdul Rahim Bey
   - **Position:** Wazir Al'Rais/Prime Minister

2. **nirahyah-noble-el-seyaraha.jpg** (or .png, .webp)
   - **Name:** Nirahyah Noble El Seyaraha
   - **Position:** Speaker of the House

3. **wisdom-shield-el.jpg** (or .png, .webp)
   - **Name:** Wisdom Shield El
   - **Position:** Assistant Qazi/Assistant Judge

4. **denzel-malik-rahman-bey.jpg** (or .png, .webp)
   - **Name:** Denzel Malik Rahman Bey
   - **Position:** Chief Shariff/Chief Sheriff

## Image Requirements

- **Format:** JPG, PNG, or WebP
- **Recommended Size:** 400x400px to 800x800px (square format works best)
- **File Size:** Keep under 500KB per image for optimal loading

## How to Add Images

1. Navigate to the `public/team/` folder in your project
2. Add the 4 images with the exact filenames listed above
3. The images will automatically appear on the website once added

## Fallback

If an image is missing or fails to load, the system will automatically use a default placeholder image.

## Where Images Appear

The team images will be displayed in:
- Homepage "Our Team" section
- Any other pages that reference the team component
